# Plank Theme - Squared Catalina

Tema Plank dengan desain mengotak dan inspirasi macOS Catalina. 

Modifikasi dari [GNOME Look](https://www.gnome-look.org/p/1308399/).

## Preview

![Preview](./assets/images/example.png)

## Instalasi

```bash
cp -r squred-catalina-black ~/.local/share/plank/themes/
```

## Uninstall

Hapus theme terlebih dahulu:

```bash
rm -rf ~/.local/share/plank/themes/squred-catalina-black
```
